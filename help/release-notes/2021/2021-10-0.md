# Cloud Manager 2021.10.0のリリースノート {#release-notes-for}

次の節では、[!UICONTROL Cloud Manager] 2021.10.0 のリリースノートの概要を説明します。

>[!NOTE]
>AEM as a Cloud ServiceのCloud Managerの最新のリリースノートについては、[現在のリリースノート ](https://experienceleague.adobe.com/en/docs/experience-manager-cloud-service/content/release-notes/cloud-manager/current#getting-access)を参照してください。

## リリース日 {#release-date}

[!UICONTROL Cloud Manager] 2021.10.0 のリリース日は 2021年10月14日（PT）です。

## 新機能 {#whats-new}

* 実稼動パイプラインを「緊急」モードで実行して、緊急デプロイメントのセキュリティテストのステップとパフォーマンステストのステップを省略できるようになりました。

* Cloud Serviceとの一貫性を保つために、既存のデプロイメントパイプラインが参照され、UIで「フルスタック」パイプラインとしてラベル付けされるようになりました。

* パイプラインカードが更新され、実稼動用と非実稼動用の両方のパイプラインを示す単一の統合面が表示されるようになりました。 ユーザーは、各パイプラインに関連付けられたアクションメニューから「実行/一時停止/再開」を直接選択できます。

* **デプロイメントマネージャー**&#x200B;の役割を持つユーザーは、UIを介してセルフサービス方式で実稼動パイプラインを削除できるようになりました。

* パイプラインエクスペリエンスの追加と編集が更新され、使い慣れた最新のモーダルを使用できるようになりました。

* Cloud Managerのユーザーは、ランディングページの右上にある「**フィードバック**」ボタンを使用して、UIから直接フィードバックを送信できるようになりました。

* 年別の SLA グラフを Cloud Manager ユーザーインターフェイスからダウンロードできるようになりました。

* コードの品質と実稼動以外のパイプラインの実行では、ビルドステップで効率的なシャロー複製プロセスを使用するようになり、特に大きなGit リポジトリを持つ顧客のビルド時間が短縮されました。

* ログインしたユーザーがブラウザーで API を試すことができるインタラクティブなプレイグラウンドに関する説明が、Cloud Manager API ドキュメントに含まれるようになりました。 詳しくは、[Cloud Manager API Playground](https://developer.adobe.com/experience-cloud/cloud-manager/reference/playground/) を参照してください。

* 「移動先」の選択オプションが無効になっている場合、プログラムカードのツールヒントがより分かりやすくなりました。 「実稼動環境は存在しません」と表示されます。


## バグ修正 {#bug-fixes}

* 内部システムから読み取ったデータが正しく入力されると、Adobe CSE （カスタマーサクセスエンジニア）が提供する無関係なデータがCloud Managerに正しく反映されない可能性があります。

* 特定の顧客状況では、ビルドのエラーの原因となっているはずのビルドステップ中にダウンロードされた無効なアーティファクトが無視されていました。
