# Cloud Manager 2021.2.0のリリースノート {#release-notes-for}

次の節では、[!UICONTROL Cloud Manager] 2021.2.0 のリリースノートの概要を説明します。

## リリース日 {#release-date}

[!UICONTROL Cloud Manager] 2021.2.0 のリリース日は 2021年2月11日（PT）です。

## 新機能 {#whats-new}

* プロジェクト作成で使用されるAEM プロジェクトアーキタイプがバージョン 25に更新されました。

* コードスキャン中に特定された非推奨の API のリストを調整し、最新 Cloud Service の SDK リリースで非推奨となった追加のクラスとメソッドが含まれるようになりました。

* 実稼動デプロイメントは、ペアのパブリッシュインスタンスとDispatcher インスタンスに並行してデプロイできるようになりました。

* Cloud Manager 用 SonarQube プロファイルが更新され、Sonar ルール `squid:S2142` が削除されました。 このプロファイルは、スレッド中断チェックと競合しなくなりました。

* ユーザーの `pom.xml` ファイルで設定されたプロパティのうち、接頭辞 sonar が付いているものは、ビルドおよび品質スキャン時のエラーを避けるために、動的に削除されるようになりました。

* Cloud Service の互換性の問題をカバーすることを目的に、コード品質ルールが追加されました。

## バグ修正 {#bug-fixes}

* CI/CD （Continuous Integration/Continuous Deployment）デプロイメントパイプラインが、パフォーマンステストステップ中にエラーが発生したロードテストを実行しているコンテナにより失敗することがあります。

* 場合によっては、例外が1つだけ発生した場合でも、実行が失敗したとロードテストコンテナが報告することがあります。 失敗は、テストプロセスを復元できない場合にのみ報告されます。

* 環境名を保存する方法の間で、文字の大文字と小文字の不一致があると、パフォーマンステストの失敗につながります。

* 一部のパイプラインの失敗が、誤ってパイプラインエラーとして報告されていました。
