# Cloud Manager 2019.6.0のリリースノート {#release-notes-for}

[!UICONTROL Cloud Manager] 2019.6.0 リリースでは、新しいコード品質ルールと新しいProduct Update ウィザードが追加されています。 詳しくは、以下の節を参照してください。

## リリース日 {#release-date}

[!UICONTROL Cloud Manager] 2019.6.0 のリリース日は 2019年6月20日（PT）です。

## 新機能 {#whats-new}

* AEM アップデートを正常に実行できるように支援する新しい製品アップデートウィザード。 詳しくは、[製品アップデート ウィザード &#x200B;](/help/product-update-wizard/overview.md)を参照してください。
* コンテンツ構造を調べるコード品質規則。 詳しくは、[&#x200B; カスタムコード品質ルール &#x200B;](/help/using/custom-code-quality-rules.md)を参照してください。
* Git プッシュの最大サイズが1 GBに増加しました。

## バグ修正 {#bug-fixes}

* 以前のエラーが原因でパイプラインを開始できない場合がありました。

## 既知の問題 {#known-issues}

* コード品質のCSV ダウンロードは、必ずしも重大度に応じてソートされるわけではありません。
* OSGi設定が&#x200B;*`config`* フォルダーの下のネストされたフォルダーに配置されている場合、*`ConfigAndInstallShouldOnlyContainOsgiNodes`* ルールで偽陽性が報告される場合があります。
