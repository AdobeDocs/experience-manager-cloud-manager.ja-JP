# Cloud Manager 2019.8.0のリリースノート {#release-notes-for}

[!UICONTROL Cloud Manager] 2019.8.0 リリースでは、一部のビルドコンテンツパッケージのサポートが追加され、ビルドパフォーマンスが向上し、様々なマイナーなバグが修正されました。

## リリース日 {#release-date}

[!UICONTROL Cloud Manager] 2019.8.0 のリリース日は 2019年8月19日（PT）です。

## 新機能 {#whats-new}

* Cloud Manager APIへの新しいコマンドラインインターフェイス（[`Adobe I/O CLI`](https://github.com/adobe/aio-cli-plugin-cloudmanager)を活用）。
* ビルドによって生成された特定のコンテンツパッケージは、スキップとして宣言され、デプロイされないことがあります。 詳しくは、[&#x200B; コンテンツパッケージのスキップ &#x200B;](/help/getting-started/project-setup.md#skipping-content-packages)を参照してください。
* ビルドコンテナのプリロードされた依存関係のセットは、一部の不要なネットワーク要求を避けるように修正されています。
* 間違って設定された特定のプログラムの概要ページのメッセージが改善されました。

## バグ修正 {#bug-fixes}

* SLA レポートにアクセスする場合、デフォルトの年が 2019 ではなく、2018 になっていました。
* 長い環境名の場合、レポート画面の環境セレクターのサイズが適切に増加していませんでした。
* ***ConfigAndInstallShouldOnlyContainOsgiNodes***&#x200B;のコード品質ルールでは、`Sling Rewriter` コンポーネントが使用されたときに偽陽性が生成されました。
* ***ConfigAndInstallShouldOnlyContainOsgiNodes***&#x200B;のコード品質ルールでは、特定の一般的でないパス構造に対して偽陽性が生成されました。
* アセットのみのお客様の場合、AEM 環境に一貫性のあるナビゲートができないことがありました。
* ブランチとプロジェクトを作成ダイアログボックスのレンダリング方法は、ブラウザによって異なります。
