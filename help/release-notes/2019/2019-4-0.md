# Cloud Manager 2019.4.0のリリースノート {#release-notes-for}

[!UICONTROL Cloud Manager] 2019.4.0 リリースには、重要な機能変更が含まれていません。 詳しくは、以下の節を参照してください。

## リリース日 {#release-date}

[!UICONTROL Cloud Manager] 2019.4.0 のリリース日は 2019年4月18日（PT）です。

## 新機能 {#whats-new}

* Cloud Manager の UI は、英語、フランス語、ドイツ語、日本語で利用できるようになりました。
* デプロイメントステップに、現在実行中のプロセスが含まれるようになりました。 つまり、バックアップの実行中に、パッケージがインストールされ、ロードバランサーが再構成されているということです

## バグ修正 {#bug-fixes}

* Dispatcher の変更をデプロイする場合のアプローチが改善され、追加の使用例が処理されるようになりました。
* 一部のインスタンスサイズタイプが環境ページに正しく表示されていませんでした。
* コード品質ルール CQBP-84依存関係は、WCM コアコンポーネントやAsset Share Commonsなどの組み込みAdobe ライブラリに対して偽陽性を生成しました。
* 詳細が使用できない場合でも、コードスキャンステップ用の詳細ボタンが有効になっていました。
* 1 分あたりのページビュー数に無効な値が指定されると表示されるエラーメッセージで、KPI の下限が正しくありませんでした。
* 実稼動以外のパイプラインのデプロイメントカテゴリが、間違って大文字になっていました。
* Git リポジトリが正しく設定されていない場合、**Overview** ページのcall to action カードに誤ったテキストが含まれていました。

## 既知の問題 {#known-issues}

* SLA 値が正しく四捨五入されない場合があります。
