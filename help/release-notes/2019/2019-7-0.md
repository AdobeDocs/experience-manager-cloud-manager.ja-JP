# Cloud Manager 2019.7.0のリリースノート {#release-notes-for}

[!UICONTROL Cloud Manager] 2019.7.0 リリースでは、バグ修正としてExperience Cloud通知の更新と機能強化が追加されています。 詳しくは、以下の節を参照してください。

## リリース日 {#release-date}

[!UICONTROL Cloud Manager] 2019.7.0 のリリース日は 2019年7月18日（PT）です。

## 新機能 {#whats-new}

実稼動デプロイメントの開始時に Experience Cloud 通知が送信されるようになりました。

## バグ修正 {#bug-fixes}

* Cloud Manager が Python および PHP ファイルに対して静的コード分析を実行する場合がありました。
* FileVault InstallHookを含むパッケージは、一貫してコード品質手順を通過しませんでした。
* 一部の組み合わせでは、コード品質の問題が正常に並べ替えられていませんでした。
* パイプライン実行ページで表示上の問題がいくつか発生していました。
* パフォーマンステストの手順は、時には基盤となるクラウドインフラストラクチャからのリソースの制約が原因で、ランダムに失敗する可能性があります。
* ネットワーク上の問題により、特定の顧客ビルドが失敗することがありました。
