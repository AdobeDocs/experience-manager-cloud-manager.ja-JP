# Cloud Manager 2019.3.0のリリースノート {#release-notes-for}

2019.3.0 リリース [!UICONTROL Cloud Manager]では、SLA レポートが前回のリリースで提供されたSystem Monitoring機能に追加されました。

## リリース日 {#release-date}

[!UICONTROL Cloud Manager] 2019.3.0 のリリース日は 2019年3月21日（PT）です。

## 新機能 {#whats-new}

* SLA レポートが Cloud Manager 内で表示されるようになりました。 詳しくは、[環境の監視](/help/using/monitoring-environments.md)を参照してください。
* 概要ページとパイプライン実行ページが再構築され、垂直方向のスペースをより有効に活用できるようになりました。
* アクティビティページで、長い日付形式が使用されるようになりました。

## バグ修正 {#bug-fixes}

* 一部の監視グラフが表示されていないことがありました。
* 監視の設定が誤っていた場合、エラーメッセージが本来よりも頻繁に表示されていました。
