# Cloud Manager 2020.1.0のリリースノート {#release-notes-for}

次の節では、[!UICONTROL Cloud Manager] 2020.1.0の一般的なリリースノートの概要を説明し、Git資格情報とログインエクスペリエンスへのアクセスに関するアップデートを追加します。

## リリース日 {#release-date}

[!UICONTROL Cloud Manager] 2020.1.0 のリリース日は 2020年1月16日（PT）です。

## 新機能 {#whats-new}

* Git の資格情報は、Cloud Manager UI 内から取得できるようになりました。 詳しくは、[Git](/help/managing-code/managing-repositories.md)へのアクセスを参照してください。
* Adobe全体での取り組みの一環として、ログイン体験とURL構造が変更されました。 古いブックマークは新しいURLにリダイレクトされます。


## バグ修正 {#bug-fixes}

* オーサーのみのトポロジへのデプロイでは、Dispatcher設定の変更はデプロイされませんでした。
* 一部の設定でコード品質のみのパイプラインを作成できないという問題を修正しました。
* 概要ページの環境サマリカードが正しく表示されないことがあるという問題を修正しました。
* パイプライン実行で、大きなトポロジでタイムアウトする可能性があります。
