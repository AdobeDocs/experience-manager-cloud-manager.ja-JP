# Cloud Manager 2020.10.0のリリースノート {#release-notes-for}

次の節では、[!UICONTROL Cloud Manager] 2020.10.0 のリリースノートの概要を説明します。

## リリース日 {#release-date}

[!UICONTROL Cloud Manager] 2020.10.0 のリリース日は 2020年10月1日（PT）です。

## バグ修正 {#bug-fixes}

* パフォーマンステストに使用されるクローラーで、特定のリソースタイプが誤って有効な Web リンクと見なされていました。

* 場合によっては、パフォーマンステストの完了ステップが正しく処理されず、手順が長時間実行されることがありました。

* 実稼動デプロイメント用にDispatcher キャッシュの無効化が設定されている場合、無効化が2回実行されることがありました。
