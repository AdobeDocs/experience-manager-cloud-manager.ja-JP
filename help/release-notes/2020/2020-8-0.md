# リリースノート（2020.8.0） {#release-notes-for}

次の節では、[!UICONTROL Cloud Manager] 2020.8.0 のリリースノートの概要を説明します。

## リリース日 {#release-date}

[!UICONTROL Cloud Manager] 2020.8.0 のリリース日は 2020年8月6日（PT）です。

## 新機能 {#whats-new}

認証にバインドされたプライベート Maven リポジトリがサポートされるようになりました。

## バグ修正 {#bug-fixes}

* 不要で望ましくないSonarQube プラグインの一部が、コード品質スキャンの一部として実行されていました。

* パイプライン実行ページで、ブランチ名の形式が正しくありませんでした。

* 1つのパブリッシュ、1つのDispatcher、コールドスタンバイオーサーを含むトポロジにデプロイすると、Dispatcherがロードバランサーから誤って削除されました。

* 完了したパイプライン実行は、完了したものとして正常に記録されなかった場合があります。 この問題により、パイプラインの新しい実行が妨げられました。

* 内部通信の問題が原因で、パイプライン実行が&#x200B;*停止*&#x200B;することがあります。

* プログラムカードのツールチップの一貫性が適切に保てていませんでした。

* **概要**&#x200B;ページで色が一致していませんでした。

* サイトパフォーマンステストで、認証のオプション使用がサポートされるようになりました。

* オーサーインスタンスのDispatcher キャッシュは、Dispatcher設定がCloud Managerを通じてデプロイされると自動的にフラッシュされます。
