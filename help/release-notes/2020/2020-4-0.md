# Cloud Manager 2020.4.0のリリースノート {#release-notes-for}

次の節では、[!UICONTROL Cloud Manager] 2020.4.0 のリリースノートの概要を説明します。

## リリース日 {#release-date}

[!UICONTROL Cloud Manager] 2020.4.0 のリリース日は 2020年4月9日（PT）です。

## 新機能 {#whats-new}

* ユーザーがプログラムを編集または切り替えられるように、ナビゲーション Cloud Managerの概要ページを変更しました。
* Cloud Manager ランディングページのプログラムカードからプログラムを編集できるようにするための変更点。
* 関連付けられた環境に対して、新しいパイプラインステータス「**パイプライン実行中**」が表示されます。
* パイプライン実行ページをわかりやすく改善しました。 この改善には、パイプライン名（実稼動以外のパイプラインのみ）とタイプの表示と、パイプラインのステータスが`In Progress`、`Canceled`、または`Failed`であるかどうかを示すバッジが含まれます。
* GIT パスワードの生成に使用されるプロセスは、基になるサービスレイヤーの問題に対して、よりレジリエンスが高くなりました。

## バグ修正 {#bug-fixes}

* 監視データは、技術的な価値のわずかな違いにもとづいて、誤った方法で表示されたり、まったく表示されたりすることがありました。
* アーティファクトのメタデータをダウンロードする際のデッドロックを回避するため、ビルドコンテナで使用される Maven の設定を更新しました。
* Assets のパフォーマンステストプロセスで、AEM パスワードを復号化できず、テストが失敗することがありました。
* スタンバイインスタンスを持つトポロジの中には、セキュリティテストが偽陽性になるものもあります。
* ステージ環境に停止したインスタンスが含まれている場合、セキュリティテスト手順が失敗することがあります。
* Experience Cloud の通知が一貫して受信されなかった。
